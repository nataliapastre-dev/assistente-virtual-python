# 🤖 Assistente Virtual em Python

Este projeto é uma **Assistente Virtual simples desenvolvida em Python**, com interface gráfica utilizando **Tkinter**.  
A aplicação é capaz de responder perguntas básicas relacionadas à programação e demonstrar conceitos iniciais de desenvolvimento com interface gráfica em Python.

Este projeto foi desenvolvido como parte do meu processo de aprendizado em programação e construção de aplicações interativas.

---

## 🚀 Funcionalidades

- Interface gráfica amigável
- Campo para envio de perguntas
- Respostas automáticas baseadas em palavras-chave
- Estrutura modular do código
- Fácil de expandir com novas funcionalidades

---

## 🛠 Tecnologias utilizadas

- **Python**
- **Tkinter** (Interface gráfica)
- **Git e GitHub** (controle de versão)

## ▶ Como executar o projeto

1. Instale o **Python 3** em seu computador

2. Clone este repositório:


git clone https://github.com/nataliapastre-dev/assistente-virtual-python.git

3. Acesse a pasta do projeto: cd assistente-virtual-python
4. Execute a aplicação: python main.py
