# 🤖 Assistente Virtual em Python

Projeto desenvolvido em **Python** com interface gráfica utilizando **Tkinter**.

## 📌 Funcionalidades

- Responde perguntas simples sobre Python
- Interface gráfica
- Sistema básico de respostas automáticas

## 🛠 Tecnologias

- Python
- Tkinter

## ▶ Como executar

1. Instale Python
2. Baixe o projeto
3. Execute:

python assistente.py
